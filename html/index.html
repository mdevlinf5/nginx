F5 Agility Labs Getting Started Doc¶

Welcome to the F5 Distributed Cloud for F5 Agility 2023

Attention

The Agility team is using this repo to document our “read-the-docs” process.

The content contained here leverages a full DevOps CI/CD pipeline and is sourced from the GitHub repository at https://github.com/f5devcentral/f5-agility-labs-xc. Bugs and Requests for enhancements can be made by opening an Issue within the repository.

Note

Each Agility lab is typically broken down by “Class” –> “Module” –> “Lab”. Each section uses “toctree” to create the index and reference the content. Using this template should give the contributer several getting-started RST examples.
